# Crazy Particles

## Project Overview

Crazy Particles is an interactive web experience built with HTML, CSS, JavaScript, Three.js, and MediaPipe. It creates a cinematic particle-based visual effect that reacts to hand gestures through your webcam. The project combines immersive motion graphics, glowing particle systems, and animated visual states to deliver a futuristic and artistic experience.

This project is ideal for anyone interested in creative coding, interactive web graphics, and browser-based visual effects.

## How to Install & Run

No complex installation is required. This project runs directly in a modern web browser.

### Step 1: Open the Project
- Download or clone the project folder.
- Open the folder in Visual Studio Code.

### Step 2: Start a Local Server
You can run the project using either of these methods:

#### Option A: Using Live Server (Recommended)
- Install the VS Code extension named Live Server.
- Right-click on the file named `index.html`.
- Choose **Open with Live Server**.

#### Option B: Using a Python Server
If Python is installed on your computer, run:

```bash
python -m http.server 8000
```

Then open your browser and visit:

```text
http://localhost:8000/
```

### Step 3: Allow Camera Access
Because the project uses hand tracking, your browser may ask for permission to use your camera. Allow access to see the interactive effects.

### Step 4: Interact
Move your hands in front of the camera to trigger the visual states and particle transformations.

## Features

- Interactive particle-based visual effects
- Real-time hand tracking using MediaPipe
- Dynamic visual states such as neutral, red, void, shrine, and purple forms
- Cinematic glowing effects with bloom lighting
- Responsive fullscreen experience for modern browsers
- Lightweight and easy to run locally
